# 毛泽东研究总导航与落稿地图

> 目标：把已经收集到的研究材料整理成一份真正能带路的总入口。
>
> 这份现在不只是本轮补料总账。
> 它承担三层用途：
> - 先告诉你整套 `research/` 该怎么读
> - 再告诉你不同问题该落到哪份主稿
> - 最后保留当前轮次最值钱的桥接证据，方便继续深挖

## 一、先看这份目录怎么分层

### A. 直接交付层

这一层是已经适合直接拿来支撑 skill 输出的主稿。
优先读这些：

- `01-writings.md`：原始文本链，适合回钉概念、句法、青年期与关键理论文本
- `02-conversations.md`：会谈与谈话，适合补现场语气、转话题方式、晚年外交语感
- `03-expression-dna.md`：表达习惯与修辞结构
- `04-external-views.md`：外部视角校正，防止只剩内部叙事
- `05-decisions.md`：关键决策的背景、选择、逻辑、结果、代价
- `06-timeline.md`：时间、节点、阶段切换
- `08-high-tension-nodes.md`：关键断裂、高张力、前后不一致处
- `10-cpc-party-history-pack.md`：官方党史口径和节点页
- `11-selected-works-full-pass.md`：整卷通读后的再提炼
- `12-selected-works-vol6-9-and-late-theory.md`：补卷、青年期、晚期理论入口

### B. 来源索引层

这一层不直接替代判断，但能帮你快速找到硬来源。

- `07-primary-source-map.md`：一手与近一手材料总索引
- `09-gap-fill-source-pack.md`：还没完全吃进主稿的硬链接与断裂链补料

### C. 原始补料层

这一层是工作包。
值钱，但默认不该先读。
只有在主稿不够细、需要追原句、追桥接证据时再回来翻。

- `13-current-round-verified-gap-pack.md`：这一轮亲手核过的可访问硬来源
- `14-current-round-merged-gap-map.md`：整套导航 + 当前轮最值钱的桥接总账

## 二、不同问题该先读哪几份

### 1. 用户要整体评估、总账、资料整理
先读：`14 -> 05 -> 06 -> 08`

先读：`06 -> 05 -> 08`

先读：`05 -> 01 -> 11`

先读：`02 -> 03 -> 01`

### 5. 用户要外部怎么评价、有没有别的视角

### 6. 用户要青年期、起步期、出厂设置
先读：`01 -> 06 -> 08 -> 11`

## 五、这一轮最该记住的总判断

这一轮真正补硬的，不是再找更多泛综述。

而是把 4 段最容易发虚的桥梁都补上了：
- `1962 回撤 -> 1963-1965 社教/四清升级`
- `1961-1964 防修焦虑 -> 中苏公开论战`
- `1965 四清路线斗争 -> 上海文宣线点火`
- `1971 内部接班链崩塌 -> 1971-1972 对美开口`

这样整条链终于能写成连续动作，而不是碎节点：

`七千人大会回撤` -> `八届十中全会再抬阶级斗争` -> `1963 社教下沉` -> `1964 桃园经验 / 第二个后十条 / 二十三条` -> `1965 海瑞罢官把路线斗争推入文艺前线` -> `1966 文革启动` -> `1971 林彪事件导致内部信任链炸裂` -> `1971-1972 借中美接近打开外部空间`

## 八、结论
