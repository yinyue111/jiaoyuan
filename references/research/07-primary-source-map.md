# 毛泽东一手与近一手材料索引图（原始文本与会谈材料）

> 目标：给后续 `思维框架 / 表达DNA / 决策史 / 晚年张力` 研究提供可核验入口。此稿只做“来源索引”，不替代主稿判断。
>
> 使用顺序：如果用户要整体整理资料、总账、阶段总览，先读 `14-current-round-merged-gap-map.md`；如果已经知道要补哪一段、缺哪种原文，再回到这份索引图找入口。
>
> 来源原则：优先 `gov.cn / cpc.people.com.cn / history.state.gov`；其次用 `marxists.org` 的公开编定本与中文数字化整理。禁用来源：知乎、微信公众号、百度百科。

## 使用标尺

- **一手**：当事时期的文稿、诗词、会谈纪要、命令、讲话原文或官方档案整理件。
- **近一手**：官方历史决议、正式编定文集、可核验公开数字化整理本。
- **可信度**：
  - **高**：官方正式文本、官方档案、官方历史决议。
  - **中高**：可核验编定本、公开数字化整理本、经典档案网站全文。
- **三种补法**：
  - **语言风格**：优先诗词、谈话、延安讲话、报刊谈话、会谈即兴答复。
  - **决策史**：优先选集、军事文集、会议讲话、FRUS、官方历史决议。
  - **晚年张力**：优先 1956—1959 治理讲话、1972—1973 中美会谈、1981/2021 官方历史评价。

## 五、重要讲话与理论文本入口

> 这一组最适合补“方法论 + 组织风格 + 说理节奏”。如果要模仿毛式判断，不先读这一组，容易只学到口号而学不到结构。

### 1. 《实践论》
- **用途**：方法论底板，解释认识从哪来、如何回到实践中检验。
- **可补判断**：调查先于判断、经验与理论关系、论证节奏。
- **更适合补**：语言风格、决策史。
- **URL**：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-193707.htm
- **可信度**：中高
- **性质**：近一手（中文数字化原文）

### 2. 《矛盾论》
- **用途**：毛式分析框架核心文本，几乎是“主要矛盾”方法的母本。
- **可补判断**：主次矛盾、矛盾特殊性、不同对象不同方法。
- **更适合补**：决策史、语言风格。
- **URL**：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-193708.htm
- **可信度**：中高
- **性质**：近一手（中文数字化原文）

### 3. 《整顿党的作风》
- **用途**：最适合看毛如何把组织问题说成“病症—纠偏—统一”的结构。
